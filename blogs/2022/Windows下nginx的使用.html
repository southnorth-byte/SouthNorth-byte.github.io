    Windows下Nginx的简单使用

    南北 2022/1/13 nginx

    在Windows下使用Nginx，我们需要掌握一些基本的操作命令，比如：启动、停止Nginx服务，重新载入Nginx等，下面我就进行一些简单的介绍。

    # 1、启动

    在nginx安装路径中使用cmd进入到D:\nginx-1.22.0>，执行

    D:\nginx-1.22.0>start nginx

或

D:\nginx-1.22.0>nginx.exe
    1
    2
    3

    注：建议使用第一种，第二种会使你的cmd窗口一直处于执行中，不能进行其他命令操作。

    如果需要特殊设置nginx的配置文件路径，可以如下执行：

    start nginx -c conf/nginx.conf

D:\nginx-1.22.0>nginx.exe -t -c conf/nginx.conf  //检查改过的配置文件是否正确

    # 2、查看进程

    可以在命令行运行tasklist命令来查看nginx进程：

    D:\nginx-1.22.0>tasklist /fi "imagename eq nginx.exe"

    # 3、退出

    nginx/Windows作为标准控制台应用运行，而不是系统服务。可以用下面的命令控制：

    nginx -s stop 快速退出
    nginx -s quit 优雅退出
    nginx -s reload 更换配置，启动新的工作进程，优雅的关闭以往的工作进程
    nginx -s reopen 重新打开日志文件
